Functies (of methods) zijn stukken code die een naam hebben gekregen. Ze worden buiten de main method geschreven. De code van een functie wordt alleen maar uitgevoerd wanneer de functie wordt opgeroepen. Als een programma (of de main method) veel dezelfde code bevat (herhaling), kan je de code die altijd hetzelfde is apart zetten in een functie en die functie telkens opnieuw oproepen.

Het definiëren van een functie.

Een functie wordt gedefiniëerd met behulp van enkele keywords:

  1. public : bepaalt of de functie overal bruikbaar is (dit kan gelimiteerd worden, hier komen we later in de cursus op terug).

  2. static : er is geen instantie van de klasse nodig om deze functie te gebruiken.

  3. void : het return type. Void (= leegte in het Engels), geeft aan dat er geen return type wordt gebruikt.

  4. de functienaam.

Bovenstaande functie geeft een stuk code aan die een naam (of een noemer) heeft gekregen, namelijk voorbeeldFunctie. Dit stuk code kan meermaals aangeroepen worden.

Een speciale functie, of method, is de main method. Deze bepaalt het startpunt van een Java programma. De definitie van deze functie heeft een vaste structuur.

Functies oproepen

Om een functie (en dus een stuk code) op te roepen, maak je gebruik van de naam van functie, gevolgd door ronde haken: .De voorbeeldfunctie aanroepen.

	public static void main(String[] args) {
		voorbeeldFunctie();
	}

Iedere keer de main functie (of main method) aangeroepen wordt (dus iedere keer je het programma start), wordt ook bovenstaande functie op zijn beurt aangeroepen. De code in de functie wordt op dat moment uitgevoerd.

Als output krijg je dat er "Hello world!" naar de console wordt geprint.

Functies met parameters

Aan een functie kan je variabelen meegeven. Deze variabelen worden meegegeven bij het oproepen van de functie. Bijvoorbeeld:

	public static void main(String[] args) {
		zegHallo("Erik");
	}

Dit zorgt ervoor dat in de volgende functie:

	public static void zegHallo(String naam){
		System.out.println("Hallo " + naam);
	}

De tekst "Erik" wordt meegegeven als naam. Deze naam parameter (of tijdelijke variabele) kan gebruikt worden in de code van de functie.

Argumenten

Iedere keer dat de functie wordt opgeroepen kan de naam parameter anders ingevuld worden:

	public static void main(String[] args) {
		zegHallo("Dirk");
		zegHallo("Beethoven");
		zegHallo("Jennifer");
	}

Iedere keer dat deze functie wordt aangeroepen, wordt er een andere naam naar de console geschreven. Dit terwijl in de code van de functie onveranderd blijft. Er worden verschillende argumenten meegegeven aan de parameter van de functie.

Functies oproepen in functies

Tot nu toe hebben we functies altijd opgeroepen vanuit de main functie (of method). Dit hoeft echter niet. Je kan perfect iedere functie oproepen vanuit een andere functie. Bijvoorbeeld:

	public static void maakKaderMetNaam(String naam){
		System.out.println("-----------------");
		zegHallo(naam);
		System.out.println("-----------------");
	}

Meerdere parameters

Je kan een functie meerdere parameters geven. Iedere keer dat de functie wordt opgroepen, moeten deze parameters ingevuld worden. Je gaat dus iedere keer argumenten moeten meegeven.

	public static void zegHalloMetLeeftijd(String naam, int leeftijd){
		System.out.println("Hallo " + naam + ", je bent " + leeftijd + " jaar oud.");
	}

Bovenstaande functie kunnen we oproepen op de volgende manier:

zegHalloMetLeeftijd("Nathan", 17){

Return types

Een return type geeft aan welk resultaat een functie teruggeeft als hij opgeroepen wordt. Tot nu toe werkten we met het void datatype. Dit geeft een leegte aan (dus niets werd er teruggegeven). We kunnen echter kiezen uit verschillende datatypes om terug te geven:

  • int : voor gehele getallen.

  • String : voor tekst.

  • double : voor kommagetallen.

  • Klassen : voor instanties van klassen (zien we later in de cursus)

  • boolean : true/false (waar / niet waar).

  • Arrays

  • ..

	public static double berekenoppervlaktevancirkel(int straal) {
		double oppervlakte = 3.1415 * straal * straal;
		return oppervlakte;
	}

In het bovenstaande voorbeeld werd het return type datatype aangepast van void naar double. Dit verplicht de code in de functie een resultaat terug te geven of te returnen. Dit wordt aangegeven met het return keyword.

public static
		double (1)
			berekenoppervlaktevancirkel(int straal) {
				double oppervlakte = 3.1415 * straal * straal;(2)
				return oppervlakte;(3)
			}
1 Het return type : dit geeft aan welk datatype de functie zal teruggeven.
2 Een berekening van een variabele.
3 Het teruggeven of returnen van een variabele.

De variabele die wordt teruggegeven moet verplicht van hetzelfde datatype zijn als het return type van de functie.

Gebruik maken van functies met return types

Als je een functie oproept die een return type heeft, kan je het resultaat van de functie opslaan in een variabele. Deze variabele kan je achteraf verder gebruiken in je code. Het resultaat van het oproepen van een functie met een return type is een tijdelijke variabele van het type van het return type.

	public static void main(String[] args) {
		double oppervlakte = berekenoppervlaktevancirkel(25);	
	}

Functies combineren

Vanuit functies kan je andere functies oproepen. Dit maakt het mogelijk in een bepaalde functie gebruik te maken van andere functies die onderliggende (complexe) zaken voor hun rekening nemen. Hierdoor kan je functies schrijven die zich focussen op hun eigen taak. Zie het voorbeeld hieronder:

	public static double berekeninhoudcylinder(int straal, int hoogte) {
		double oppervlakte = berekenoppervlaktevancirkel(straal);
		double inhoud = oppervlakte * hoogte;
		return inhoud;
		
	}
	
	public static double berekenoppervlaktevancirkel(int straal) {
		double oppervlakte = 3.1415 * straal * straal;
		return oppervlakte;
	}
